Fandom Wiki Web Scraping Assignment

For this assignment, I chose the Harry Potter Wiki. I am interested in scraping data about characters, including their names and links to their individual pages.

I chose this fandom because I have always loved the Harry Potter serires and it has a large number of well-documented characters, which makes it useful for analyzing patterns such as character distribution and relationships. This type of data could be useful for researchers studying storytelling, character development, or fan engagement.

Before scraping, I checked the robots.txt file to make sure scraping is allowed:
https://harrypotter.fandom.com/robots.txt

The website allows access to the pages used in this assignment.

The data collected is stored in a CSV file for easy use and analysis.
