# CLI Movie Data Entry

This Python script allows users to enter movie information through the command line and saves the data to a text file.

## Features
- Prompts the user to enter movie information
- Allows multiple movie entries
- Saves the movie data to a file
- Prints a message confirming the data was saved

## How It Works
1. The program asks the user to enter movie details.
2. The user can continue adding more movies.
3. The data is saved to a file called `movies.txt`.
4. A confirmation message is displayed.
